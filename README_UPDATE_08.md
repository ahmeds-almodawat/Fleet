## Important notes

This update does not change Supabase, storage buckets, Edge Functions, secrets, or migrations.

Some values can still appear in English if they are user-entered or stored in the database, for example:

- Vehicle names/codes
- Destination names
- Driver/user names
- Custom notes/descriptions
- Existing audit-event details
- Existing notification body text already saved before this update

Those are data values, not UI labels. They will display in the language entered by the user or generated by older database/function logic.
